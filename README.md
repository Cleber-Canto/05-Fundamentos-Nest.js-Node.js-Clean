## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov


:

    npm run build
        Descrição: Compila o projeto Nest.js.
        Como executar: npm run build

    npm run format
        Descrição: Formata automaticamente os arquivos TypeScript usando o Prettier.
        Como executar: npm run format

    npm start
        Descrição: Inicia o servidor Nest.js em modo padrão.
        Como executar: npm start

    npm run start:dev
        Descrição: Inicia o servidor Nest.js em modo de desenvolvimento com observação de alterações.
        Como executar: npm run start:dev

    npm run start:debug
        Descrição: Inicia o servidor Nest.js em modo de depuração com observação de alterações.
        Como executar: npm run start:debug

    npm run start:prod
        Descrição: Inicia o servidor Nest.js em modo de produção a partir dos arquivos compilados.
        Como executar: npm run start:prod

    npm run lint
        Descrição: Executa o ESLint para verificar e corrigir problemas de linting nos arquivos TypeScript.
        Como executar: npm run lint

    npm test
        Descrição: Executa os testes usando o Vitest.
        Como executar: npm test

    npm run test:watch
        Descrição: Executa os testes usando o Vitest em modo de observação de alterações.
        Como executar: npm run test:watch

    npm run test:cov
        Descrição: Executa os testes usando o Vitest com relatório de cobertura de código.
        Como executar: npm run test:cov

    npm run test:debug
        Descrição: Executa os testes usando o Vitest em modo de depuração.
        Como executar: npm run test:debug

    npm run test:e2e
        Descrição: Executa testes de ponta a ponta usando o Vitest com uma configuração específica (vitest.config.e2e.ts).
        Como executar: npm run test:e2e

    npm run test:e2e:watch
        Descrição: Executa testes de ponta a ponta usando o Vitest com observação de alterações e configuração específica.
        Como executar: npm run test:e2e:watch


```
